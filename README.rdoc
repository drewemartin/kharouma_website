==Welcome to Kharouma's Website

This is a pro-bono project for a friend that is starting a sustainable palm oil venture in East Africa.


==Functionality Highlights

* User authentication and password reset (powered by Sorcery gem)

* Action mailer for password reset (supported by Mandrill)

==Specs

* Made with Ruby on Rails(infrastructure) 

* TDD development for data model, crafted using Rspec, FactoryGirl and Faker
